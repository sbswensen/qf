QF Page

This page contains all the links to the features of QF.
The links here are only accessible by the administrator.

Users And Roles

  1. Roles
    For the creation, listing and deletion of roles.
  2. Users And Roles Association
    A listing of the users and the roles attached to the users. It further provides links to edit the association.

Document Structures

  1. New Document Structure
    A document structure is a collection of data that leads to creation of form pages. This creates a new document structure.
  2. List Document Structures
    This list the document structures and provides links to create, view document structure, view created lists and delete document structures.
    In the view page of document structures you have links to edit the document structure, edit its permissions and add approvals framework to it.

Buttons

  1. Create Button
    This creates a new button which is shared across document structures. The button would point to a link which would run the actions of the button. The administrator would write the receiving link.
  2. List Buttons
    This list the buttons created and some details about the buttons. It also contains links to delete buttons.
